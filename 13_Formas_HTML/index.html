Ejemplos de elemntos de formulatios

Formas

<form> Es el contenedor del formulario que reune los elementos que reciben el input del usuario. Determina las acciones al enviarlo y la forma de hacerlo. Docs

Input

<input> es un elemento de entrada de datos genérico que adopta la forma necesaria según su tipo. Uno de sus atributos más importantes es name que lo identifica dentro de la forma y en la solcitud que se envía al confirmar. Docs

Algunos "type" comunes son:

  • "text" que muestra una entrada para que el usuario teclee texto.

    <input type="text" /> ->

  • "checkbox" muestra una caja de selección con valor binario, para darle un valor se usa un nodo label

    <input type="checkbox" /><label>Funcional?<label> ->

  • "radio" similar a checkbox con la diferencia de que genera una familia de selectores donde solo uno puede estar seleccionado.

    <input type="radio" name="numero" value="1" \><label>Uno<label>
    <input type="radio" name="numero" value="2" \><label>dos<label> ->

  • "submit" Genera un botón que ejecuta la acción principal, "submit", de la forma.

    <input type="submit"\> ->

Ejemplo

Este código genera la forma que vemos más abajo.

<form>
<label>Nombre</label> <input type="text" name="name">
<label>Edad</label> <input type="number" name="age">
<label>Correo Electrónico</label> <input type="email" name="email">
<input type="checkbox" name="tycs"> <label>Acepta términos y condiciones</label>
<input type="submit">
</form>
